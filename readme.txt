A full-suite WooCommerce extension to manage AI-assisted print order quotations, PDF generation, and print house selection using ACF and Google Sheets.

== Description ==

**PrintPrice Pro** is a custom plugin designed for [printprice.pro](https://printprice.pro), allowing users to:
- Interact with an AI chatbot to define print specifications.
- Automatically calculate estimates from Google Sheets.
- Select compatible print houses.
- Store and export orders with ACF fields.
- Generate downloadable PDF summaries.

Features:
* WooCommerce integration.
* Google Sheets data sync.
* AI Engine compatibility.
* Multilingual support (EN, FR, DE, LT).
* Print-ready PDF generation using TCPDF.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate through the 'Plugins' menu in WordPress.
3. Make sure ACF Pro and WooCommerce are installed and active.

== Frequently Asked Questions ==

= Can I customize the PDF layout? =
Yes, edit the TCPDF template located in `includes/pdf/`.

= How does it connect with Google Sheets? =
Use your API key in the settings and set your Google Sheet ID and ranges in the configuration.
